MY PROJECTS

Project: AI WhatsApp Business Assistant (Generative AI / RAG)
This is a Retrieval-Augmented Generation system I built to give businesses an intelligent WhatsApp assistant powered by their own documents and knowledge bases. The system uses semantic search so it retrieves the most relevant information before generating a response, meaning the answers are grounded in real business data rather than hallucinated. It supports conversation memory so the assistant remembers what was discussed earlier in a chat thread. Technologies used: Python, LLMs, LangGraph, Vector Database, PostgreSQL, Firebase.

Project: African Skin Disease Detection System (Healthcare AI)
This is a machine learning research project focused on improving skin disease detection for African patients. Most existing skin disease AI models are trained on datasets dominated by lighter skin tones, which means they perform poorly on darker skin. My work at DISAL tackled this representation problem using MedGemma-based models and African-specific healthcare data. This project sits at the intersection of computer vision, healthcare equity, and responsible AI. Technologies used: Deep Learning, Computer Vision, MedGemma, Python.

Project: TweetEval NLP Classification Model (Natural Language Processing)
I built a three-class NLP text classification model that categorizes social media content into safe content, neutral content, and offensive or hate speech. This kind of content moderation model is important for building safer online spaces. Technologies used: Python, Hugging Face Transformers, NLP, Machine Learning.
